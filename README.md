# codingTrackerServer
This is the server for the codingTracker app.

## Description
Using a REST api, it save in a sqlite3.0 db the time spend coding in
language that are tracked by codingTracker app.

## Getting Started

### Dependencies
poetry
aiohttp
pyyaml

### Installing
poetry install

### Executing program
Run the server:
	poetry shell
	python3 ./codingTrackerServer/server.py
